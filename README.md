# IPCalculate
Program mencari IP mahasiswa. 
Mencari IP mahasiswa, dengan memasukkan mata kuliah, sks dan berapa nilai mahasiswa tersebut.

Example : 

---Input---

Nama  : Rezky Revindo

Kelas : IT-41-01\n

Banyak Mata Kuliah : 2

Nama Mata Kuliah 1 : Kalkulus

Nama Mata Kuliah 2 : Matematika\

Data Nilai

Kalkulus 

SKS : 2 

Nilai : 70 

Matematika

SKS : 4

Nilai : 60

---Output---

Data Mahasiswa

Nama : Rezky Revindo

Kelas : IT-41-01

IP : 63(B)
